# Scanner-like-lex

### 使用方法
1. 在一開始define的地方寫上input.txt的檔名
```cpp=
#define filePath "hw1_input.txt"
```
2. 將scanner.cpp檔跟input.txt檔放入同一個資料夾中
3. 建議使用visual studio code去編譯，直接按run code即可
